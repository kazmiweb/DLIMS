# DLIMS

### Install Django
```bash
python -m pip install Django
```

### Install Requirements
```bash
pip install -r requirements
```

### Run
```bash
python manage.py runserver
```

### Admin
- URL: localhost:8000/admin
- Username: root
- Password: root

### URLS
http://localhost:8000/license/verify
http://localhost:8000/license/verify/save
